| #### Wortspiel — Memory Game <br><br> **File Structure** <br> `wortspiel/` <br> `├── docker-compose.yml` (Compose file to install the game easily) <br> `├── sql/database.sql` (SQL Script to create the database) <br> `└── web/` <br> `    ├── index.html` (The game) <br> `    ├── css/style.css` (The styles) <br> `    ├── js/main.js` (Instructions / functions) <br> `    └── php/api.php` (API that reads vocabulary from MySQL) <br><br> **Requirements** <br> If developing the web app in a container, you will need: Docker & Docker Compose installed. <br><br> **Installation** <br> 1. **Install the game**: From the terminal, run `docker-compose up -d --build` at the root of the project. <br> 2. **Configure api.php**: Edit the constants in `api.php`: `DB_HOST`, `DB_USER`, `DB_PASS`, `DB_NAME`. <br> 3. **Deploy**: Access `http://your-server/wortspiel/index.html`. <br><br> --- <br><br> **API Reference** <br> `GET api.php?action=pairs&count=8` <br> Returns N random pairs. <br> `{"success": true, "pairs": [{ "id": 1, "aleman": "sein", "espanol": "ser, estar" }]}` <br><br> --- <br><br> **Game Rules** <br> 1. All cards start face down. <br> 2. Click a card to flip it over. <br> 3. If the cards match, they remain uncovered. <br> 4. You win when you match all the cards. |
